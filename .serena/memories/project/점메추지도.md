# 점메추 지도 서비스 정리

## 개요
- **서비스명**: 점메추 지도
- **URL**: https://jummechr.vercel.app
- **GitHub**: https://github.com/hwks0048-create/jummechr
- **목적**: 직장인 점심 메뉴 고민 해결 — 위치 기반 주변 맛집 4곳 즉시 추천
- **타겟**: 점심시간 1시간 안에 식사해야 하는 직장인

## 기술 스택
- **프레임워크**: Next.js 16.1.6 (App Router, TypeScript)
- **스타일**: Tailwind CSS v4 + 인라인 스타일
- **아이콘**: lucide-react
- **호스팅**: Vercel (무료)

## 사용 API
| API | 용도 | 환경변수 |
|-----|------|---------|
| 카카오 로컬 API | 500m 내 맛집 검색 (좌표 기반) | `KAKAO_REST_API_KEY` |
| 네이버 지도 NCP | 지도 표시 | `NEXT_PUBLIC_NAVER_MAP_CLIENT_ID` = jdpakm3g5n |

- 카카오 앱: developers.kakao.com (앱명: 점메추지도)
- 네이버 지도: console.ncloud.com

## 핵심 기능
1. 브라우저 Geolocation으로 현재 위치 감지
2. 카카오 로컬 API로 500m 내 음식점 75개(5페이지) 검색
3. **한식 / 중식 / 일식 / 양식** 4카테고리 각 1곳 랜덤 추천 (2x2 그리드)
4. 중식/일식 버킷이 5개 미만이면 키워드 폴백("중국집"/"일식당", 1500m)으로 보완
5. 3번까지 이전 가게 제외(exclude 파라미터) → 4번째부터 리셋
6. 네이버 지도에 마커 4개 + fitBounds로 한눈에 표시 (카테고리별 색상)
7. 카드/마커 클릭 → 카카오맵 플레이스 페이지 연결
8. GA4 연동 (G-RWMLN28D7H)

## 파일 구조
```
app/
  api/restaurants/route.ts  ← 카카오 API 호출, 카테고리 분류
  layout.tsx                ← SEO 메타데이터, Schema.org JSON-LD
  page.tsx                  ← 메인 UI (뽑기 버튼, 카드, 지도)
  sitemap.ts / robots.ts    ← SEO
components/
  NaverMap.tsx              ← 네이버 지도 컴포넌트
public/
  og.jpg                    ← OG 이미지 (1200x630)
```

## SEO
- 타겟 키워드: 점메추, 점심 메뉴 추천, 점심 뭐먹지, 직장인 점심
- Schema.org: WebApplication + FAQPage
- Google Search Console 등록 완료 + 사이트맵 제출
- 네이버 서치어드바이저 등록 완료 + 사이트맵 제출
- OG 이미지 적용 (og.jpg, 1200x630)

## 무료 한도 (DAU 기준)
- 카카오 API: ~10,000 DAU (300,000 calls/일 ÷ 7 calls/뽑기 ÷ 4회/인) — 중식 폴백 시 최대 7호출
- 네이버 지도: ~6,000~7,000 DAU (200,000 calls/월) ← 병목
- Vercel: ~17,000 DAU (100GB 대역폭/월)
